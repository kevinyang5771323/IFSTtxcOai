# 项目简介 java865

本项目是一个基于Spring Boot的图书管理系统，旨在通过实践作业加强对后端和前端技术的理解和应用。

## 技术栈

- **后端**：Spring Boot、Spring Framework、MyBatis、MySQL
- **前端**：HTML、CSS、JavaScript、Vue.js

## 功能模块

- 用户管理：注册、登录、信息修改
- 图书管理：图书检索、借阅、归还、图书信息维护
- 系统管理：权限控制、日志管理

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- **操作系统**：Windows/Linux/MacOS
- **数据库**：MySQL 5.7+
- **Java环境**：JDK 1.8+

## 部署步骤

1. 安装并配置好Java环境
2. 安装MySQL数据库，并根据项目要求初始化数据库
3. 克隆或下载项目代码
4. 编译并运行项目
